API (Application Programming Interface)
	응용프로그램에서 다른 소프트웨어와 상호 작용을 할 수 있도록 제공되는 일련의 규칙과 명령어 집합.
	다른 서비스. 라이브러리, 운영 체제 또는 플랫폼과 통신하기 위해 사용.
	프로그래머에게 특정 서비스 또는 라이브러리와 상호 작용할 수 있는 방법을 제공
	HTTP 요청을 통해 데이터를 전송하고 받는 형태로 사용
	

SDK (Software Development Kit)
	특정 소프트웨어 또는 하드웨어 플랫폼에 대한 개발 도구와 리소스 집합.
	SDK는 개발자가 특정 플랫폼 또는 서비스를 활용해서 응용 프로그램을 만들때 
	필요한 도구, 라이브러리, 예제코드 및 문서를 포함한다.
	일반적으로 API 호출을 추상화 하고 개밣자에게 보다 쉬운 방법으로 상호작용을 할 수 있는 인터페이를 제공

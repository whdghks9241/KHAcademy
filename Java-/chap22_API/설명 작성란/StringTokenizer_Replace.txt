StringTokenizer 와 Replace
공통점 : 둘 다 문자열을 처리하는 Java에서 사용되는 기능.

*파싱 : 데이터를 분해 분석하여 원하는 형태로 조립하고 다
시 빼내는것

StringTokenizer 
	문자열을 특정 구분자를 기준으로 토큰으로 분리하는데 사용
	입력문자열을 하핑하거나 특정 형식의 문자열을 분해하는데 이용한다.
	CSV 파일(엑셀파일) 에서 데이터를 읽을 때나 특정 혁식의 문자열을 파싱할 때 사용할 수 있음.
	예제 코드 : 
		String input = "apple, banana, grape"
		StringTokenizer  t= new StringTokenizer(input, ",")
		// "," 구분자를 사용하여 분자열을 분리하고 각각 다톤 토클ㄴ 출력
		

replace 
	메서드는 문자열 내에서 특정 문자열 또는 문자를 다른 문자열 또는 문자로 교체하는데 사용
	문자열에서 특정 패턴을 찾아서 변경할 때 유용
	예를 들어 문자열 내의 모든 공백을 다른문자로 변경할때 사용할 수 있다.
	예제 코드 :
		String input = "Hello World!;
		String reInput = input,replace(" ", " -")
		system.out.purint(input)

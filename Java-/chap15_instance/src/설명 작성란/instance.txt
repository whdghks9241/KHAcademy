인스턴스
	참조변수를 통해서 다룰 수 있음
	참조변수의 타입은 인스턴스의 타입과 일치해야함.
	객체 : 클래스를 기반으로 만들어진 모든 것
	인스턴스 : 클래스로부터 생성된 개별 객체
	
		예제코드 : 
		
		// 클래스의 객체로 참조하기 위한 참조변수 선언
			클래스명 변수명 ;
		
		// 클래스의 객체를 생성 후 객체의 주소를 참조변수에 저장
			변수명 - new클래스 ();
		
		// 인스턴스를 참조하기 위한 tv선언
		// 메모리공간에 참조변수 tv를 위한 공간 선언!
		// 인스턴스 생성되지 않은 상태이므로 참조변수로 할 수 있는 것은 없으며 선언만 된 것 일 뿐
			Tv tv;
		
		// 클래스의 객체를 생성 후, 객체의 주소를 참조변수에 저장 
		// tv인스턴스를 생성한다음, 생성된 tv 인스턴스 주소를 참조변수 tv에 저장
		// 연산자 new에 의해 tv클래스의 인스턴스가 메모리 빈 공간에 생성
		// 대입연산자(=)에 의해 생성된 객체의 주소값이 참조변수 tv에 저장
		// 참조변수 tv를 통해 tv 인스턴스에 접근할 수 있음
		// 인스턴스를 다ㅏ루기 위해서 ㅊ마조변수는 반드시 필요함
			tv = new Tv();

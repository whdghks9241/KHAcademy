데이터를 저장하기 위해 이름을 할당받은 메모리 공간을 의미
메모리 공간 저장할 수 있는 공간을 의미하며 이렇게 저장된 값은 변경될 수 있다


변수종류




1. 맴버변수(Member Variables) :
	클래스 또는 객체에서 일부로 선언되는 변수
	객체의 상태를 나타내고, 클래스의 모든 메서드에서 사용할 수 있다.
	객체가 생성될 때 초기화 되고 객체의 수명과 동일한 수명을 가짐
	예제 코드 :
		public class MyClass {
			int memberVar; // 맴버변수
		}
		
2. 정적변수(Static variables) :
	클래스에서 선언언되며 모든 객체가 공유
	클래스 이름으로 직접 접근하여, 클래스가 로드될 때 초기화
	객체의 수명과 무관, 프로그램이 종료될 때 존재
	예제 코드 :
		public class MyClass {
			static int staticVar; // 정적변
		}
		
3. 인스턴스 변수(Instance Variables)
	객체마다 별도로 가지는 변수 
	객체가 생성될 떄 초기화되며, 객체의 수명과 동일한 수명을 가짐.
	예제 코드 : 	
		public class MytClass {
			int instanceBar // 인ㅅ흐턴스 변수;
		}
		
4. 참조변수 (Reference Variables) :
	객체를 가리키는 참조를 저장하는 변수
	객체의 타입에 따라 다양한 참조변수를 선언할수 있다.
	예저코드 :
	
		MyClass myC; // MyClass객체를 가리키느  ㅊ마조변수
		String str; // String객체를 가리키는 변수
		
		
5. 지역변수(Local Variables_; 
	메서드나 블록 내에서 선언되며, 해당 블록 내에서만 사용가능한
	매서드나 블록이 실행될 때 생성되며, 실행이 종료되면 사라진다
	예쩨코드,
		public void myMoethod() {
			int localVar; // 지역변수
			localVar 사용가능
		}
		local 사용가능
		
		for (int i = 1; i < 0; t++) {
			// i 사용 가능
		}
		
		i 사용불가능
		
		
6. 파라미터 변수(Parameter Variables) : 
	매서드나 생성자에 전달되는 값 또는 변수
	해서드 내에서 작업을 수행하는 데 필요한 데이터를 전달하는데 사용
	메서드를 호출할 때 인자(argument)로 전달
	파라미터 : 메서드 정시에 선언한 변수, 메서드 내에서 자용되는 벼수 이름
	인자 : 메서드 호출 시에 전달하는 실제 값,. 메서드 호출할 때 사용되는값	
	예제 코드 :
		public int add (int a, int b) { // a와 b는 파라미터
			return a+b;
		}
		
		public static void main(String[] args) {
			myClass s = new MyClass();
			
			// 메서드 호출 : 인자(실제) 값을 전달
			int result = s.add(5.3) 
		}
